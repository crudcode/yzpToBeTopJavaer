##### （1）策略模式 （strategy pattern）
- 原理：策略模式定义了算法族，分别封闭根活佛，让它们之间可以互相替换，此模式让算法的变化独立于使用算法的客户
- 使用场景举例：如有许多鸭子类型，每种鸭子都有不同的特点，如有的会飞有的不会飞。假如对每一种鸭子对象都定义特点，
就要 写很多相同的代码。这里就要把这些 特点都抽取出来，当有新类型鸭子时，只要将相同的特点赋予新的鸭子，达到复用。
